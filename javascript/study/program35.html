🖱 프로그래머스 1단계 : 중복된 문자 제거 🖱

📖 문제 설명 📖

문자열 my_string이 매개변수로 주어집니다. my_string에서 중복된 문자를 제거하고 하나의 문자만 남긴 문자열을 return하도록 solution 함수를 완성해주세요.

🧸 제한 사항 🧸

1 ≤ my_string ≤ 110
my_string은 대문자, 소문자, 공백으로 구성되어 있습니다.
대문자와 소문자를 구분합니다.
공백(" ")도 하나의 문자로 구분합니다.
중복된 문자 중 가장 앞에 있는 문자를 남깁니다. 

function solution(my_string) {
    let str = my_string.split('');
    
    let answer = str.filter((v, i) => str.indexOf(v) === i).join("")
    
    return answer;
}
// people 이라면 peol을 리턴합니다.

🧸 문제 풀이 🧸

1. 먼저 문자열을 split으로 쪼개어 하나의 배열을 만들어줍니다.
2. 그 후 filter와 indexOf를 이용하여 값과 인덱스가 같을 경우 join 시키도록 하였습니다. 즉 중복된 값은 인덱스가 하나로 정의되지 않으므로 자연스럽게 뒤의 값은 제거됩니다.
3. 중복된 문자가 제거된 값을 변수 answer에 저장하여 리턴합니다.

🧸 다른 사람의 풀이 🧸

function solution(my_string) {
    return [...new Set(my_string)].join('');
}

Set : 자바스크립트에서 중복되는 데이터를 제거해줍니다. 중복된 값이 있으면 한개만 유일하게 남겨주는 것이지요! 이렇게 새로운 메서드를 하나 알게되네요~!
