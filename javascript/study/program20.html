🖱 프로그래머스 1단계 : 하샤드 수 🖱

📖 문제 설명 📖

양의 정수 x가 하샤드 수이려면 x의 자릿수의 합으로 x가 나누어져야 합니다. 예를 들어 18의 자릿수 합은 1+8=9이고, 18은 9로 나누어 떨어지므로 18은 하샤드 수입니다. 자연수 x를 입력받아 x가 하샤드 수인지 아닌지 검사하는 함수, solution을 완성해주세요.

🧸 제한 사항 🧸

x는 1 이상, 10000 이하인 정수입니다. 

function solution(x) {
    let sum =0;
    let arr = String(x).split("");

    for(let i=0; i<arr.length; i++){
        sum += Number(arr[i]);
    }

    return (x % sum === 0) ? true : false;
}

// 입력값이 10일 때 모든 자릿수의 합은 1이다. 하샤드 수를 구하는 식에 대입해보았을 때 10 / 1 == 0으로 딱 떨어지므로 true 반환시킨다.
// split :  String 객체(문자열)를 구분자를 사용하여 여러 개의 문자열로 나누고 그 값을 배열로 반환시킵니다.

🧸 문제 풀이 🧸

1. String()을 통해 x를 문자열로 만들어 줍니다. 그 뒤 split()를 통해 하나씩 쪼개어줍니다.
2. 반복문을 통해 자릿수 만큼 반복 시킨후 Number()로 문자열을 정수로 만들어줍니다.
3. sum 변수에 정수로 변환한 값을 넣어준 후 총 합을 구합니다. (10이면 1 + 0 이므로 1)
4. 리턴으로 만약 x의 값을 sum의 값들로 나누었을 때 딱 떨어진다면 하샤드 수 이므로 ture를 반환합니다. 나머지 값이 생길 경우 false를 반환합니다.
