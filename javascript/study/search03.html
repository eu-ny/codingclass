알파벳 버튼 작업이 끝나면 속성 리스트를 작업해줍니다.

✏️ 리스트 보기 
<li><strong>accent-color</strong> : accent-color는 요소의 강조 색상을 지정합니다.</li>
<li><strong>align-content</strong> : align-content는 콘텐츠 사이와 콘텐츠 주위 빈 공간을 플렉스 박스'의 교차축 또는 그리드의 블록 축을 따라 배치하는 방식을 결정합니다.</li>
<li><strong>align-items</strong> : align-item은 콘텐츠 아이템의 내부 상하관계 정렬 상태를 설정합니다.</li>
<li><strong>align-self</strong> : align-self는 개별적 콘텐츠 아이템의 정렬 상태를 설정합니다.</li>
<li><strong>all</strong> : all은 요소의 모든 속성을 초기화합니다. 또는 상속을 설정합니다.</li>
<li><strong>animation</strong> : animation은 애니메이션과 관련된 속성을 일괄적으로 설정합니다.</li>
<li><strong>animation-delay</strong> : animation-delay는 애니메이션 지연 시간을 설정합니다.</li>
<li><strong>animation-direction</strong> : animation-direction는 애니메이션 움직임 방향을 설정합니다.</li>
<li><strong>animation-duration</strong> : animation-duration는 애니메이션 움직임 시간을 설정합니다.</li>
<li><strong>animation-fill-mode</strong> : animation-fill-mode는 애니메이션이 끝난 후의 상태를 설정합니다.</li>
<li><strong>animation-iteration-count</strong> : animation-iteration-count는 애니메이션의 반복 횟수를 설정합니다.</li>
<li><strong>animation-name</strong> : animation-name은 애니메이션 Keyframe 이름을 설정합니다.</li>
<li><strong>animation-play-state</strong> : animation-play-state는 애니메이션의 진행 상태를 설정합니다.</li>
<li><strong>animation-timeline</strong> : animation-timeline은 요소에 적용할 스크롤 애니메이션을 설명하는 하나 이상의 @scroll-timeline at-rules 이름을 지정합니다.</li>
<li><strong>animation-timing-function</strong> : animation-timing-function은 애니메이션 움직임의 속도를 설정합니다.</li>
<li><strong>appearance</strong> : appearance는 webit 계열의 브라우저의 type="search" 필드의 둥근 테두리 값이나 reset 효과를 나타내는 버튼을 삭제하는 데 사용할 수 있습니다.</li>
<li><strong>aspect-ratio</strong> : aspect-ratio는 요소의 크기를 비율대로 조정할 수 있게 합니다.</li>

#4. JAVASCRIPT : 스크립트 짜기

charAt()와 반복문, 조건문을 통해 스크립트를 완성해보자!
코드의 주석과 함께 스크립트를 이해해보도록 합니다:)

✏️스크립트 보기

//선택자
const searchKeyword = document.querySelectorAll(".search__info .keyword span");  //알파벳
const searchList = document.querySelectorAll(".search__list ul li");  //목록 리스트
const searchInfo = document.querySelector(".search__info .num");      //갯수

//전체 갯수 구하기
searchInfo.textContent = searchList.length;

//searchBox 검색 영역
searchKeyword.forEach(el => {
    el.addEventListener("click", () => {
        const searchWord = el.innerText;  // 사용자가 클릭한 알파벳 / searchWord에 클릭한 알파벳 데이터 저장

        searchList.forEach(el => {
            const cssName = el.querySelector("strong").innerText; //CSS 속성 텍스트 / cssName에 셀렉터한 요소 데이터 저장

            //알파벳 첫글자와 css속성의 첫글자가 같은 것만 출력된다.
            if(searchWord.charAt(0) === cssName.charAt(0)){
                el.classList.remove("hide");
            }else {
                el.classList.add("hide");
            }
        });
    });
});

! 검색 영역이 작동되는 과정 알아보기 !
***charAt(검색값) 메서드는 사용자가 검색한 문자열이 상수에 저장한 데이터 내에 포함되었을 때 숫자를 반환하게 됩니다.***
1. forEach문을 통해 사용자가 알파벳을 클릭한 경우 반복문과 조건문을 실행시킵니다.
2. classList를 통해 hide라는 클래스를 추가해주고 charAt()을 이용하여 css속성의 첫글자와 알파벳이 동일한 리스트는 hide가 지워져 출력되도록 합니다.
(→ css 속성값으로 hide에게 display: none;을 주었기 때문에 값이 동일하지 않은 경우 보이지 않게 된다!)
