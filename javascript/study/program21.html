🖱 프로그래머스 1단계 : 정수 제곱근 판별 🖱

📖 문제 설명 📖

임의의 양의 정수 n에 대해, n이 어떤 양의 정수 x의 제곱인지 아닌지 판단하려 합니다. n이 양의 정수 x의 제곱이라면 x+1의 제곱을 리턴하고, n이 양의 정수 x의 제곱이 아니라면 -1을 리턴하는 함수를 완성하세요.

🧸 제한 사항 🧸

n은 1이상, 50000000000000 이하인 양의 정수입니다. 

function solution(n){
    let result = 0;
    let num = Math.sqrt(n);
 
    if (num % 1 == 0){
      result = ( num + 1 ) * ( num + 1 );
    } else {
      result = -1;
    }
 
    return result;
}

// Math.sqrt(n) : n의 제곱근을 구하는 수학 메서드입니다. 만약 음수알 경우 NaN을 반환합니다.
// n = 121일 때 144를 반환 n = 3일 땐 -1 반환.

🧸 문제 풀이 🧸

1. 변수를 만들어서 제곱근을 구하는 식을 값으로 넣어 줍니다.
2. 조건문( 정수는 1로 나누었을 경우 항상 나머지가 0이 되는 속성을 이용해서 주어진 숫자가 정수인지 체크 )을 통해 만약 num이 양의 정수인 경우 num + 1의 제곱을 구하여 변수에 값을 저장합니다.
3. 만약 아닐 경우 -1을 반환하도록 합니다.

🧸 다른 사람의 풀이 🧸

function solution(n) {
    var answer = 0;
    let sqrt = Math.sqrt(n);
    if (sqrt % 1 !== 0) {
        answer = -1;
    } else {
        answer = Math.pow(sqrt+1,2);
    }
    return answer
}
