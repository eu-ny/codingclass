자바스크립트 : 문자열 메서드 : includes()

이번 시간에는 문자열 메서드인 includes()에 대해 알아보겠습니다! 😁

#1. includes( )

includes() 메서드는 문자열 포함 여부를 검색하여, 불린(true, false)을 반환합니다. 즉 문자열이 특정 문자열을 포함하는지 확인하는 메서드라 볼 수 있습니다!

! includes( ) 사용 방법 !
string.includes( searchString, length )
"문자열".includes(검색할 문자열);
"문자열".includes(검색할 문자열, 검색을 시작할 위치); == indexOF()와 거의 같다. 

const str1 = "javascript reference";

const currentStr1 = str1.includes("javascript");    // true
const currentStr2= str1.includes("j");              // true
const currentStr3= str1.includes("b");              // false : b는 포함되어 있지 않으므로 false
const currentStr4= str1.includes("reference");      // true
const currentStr5 = str1.includes("reference", 1);    // true
const currentStr6 = str1.includes("reference", 11);   // true
const currentStr7 = str1.includes("reference", 12);   // false : 문자열 자체에 위치값인 12가 없기 때문에 false

! point !
'abzcd'.includes( 'z' )
includes() 메서드가 문자열인 'abzcd'에서 z를 포함하는지 검사한다. 위 식은 z를 포함하므로 true를 반환한다!
'abZcd'.includes( 'z', 3 )
includes() 메서드가 문자열인 'abZcd'의 3번째 위치부터 z를 포함하는지 검사한다. includes()는 대소문자를 구분하기 때문에 z를 포함하지 않으므로 false를 반환한다!
