자바스크립트 : 배열 메서드

지금부터 자바스크립트의 배열 메소드인 join() / push() / pop()에 대해 알아봅시다!
배열 메서드들을 이해하게 되면 배열을 좀 더 효율적으로 사용할 수 있게됩니다 :)

1. join( )

join( )은 배열을 변경하지 않고 참조만 하는 메서드로 배열의 모든 요소를 하나의 문자열로 반환합니다. 

//01
const arrNum = [100, 200, 300, 400, 500];
const text1 = arrNum.join('');  //배열의 요소를 전부 이어서 불러온다.
const text2 = arrNum.join('   '); //한 칸을 띄우면 join이 여백을 인식해서 한 칸씩 띄워서 표현된다.
const text3 = arrNum.join('★'); //별을 넣게 되면 join이 별을 인식해서 불러올때도 별이 각 배열값 사이에 같이 나온다.
const text4 = arrNum.join('-'); //-을 넣게 되면 join이 -을 인식해서 불러올때도 -이 각 배열값 사이에 같이 나온다.
표로 확인해보자!
번호 기본값 메서드 리턴값
1 [100, 200, 300, 400, 500] join('') 100200300400500
2 [100, 200, 300, 400, 500] join('  ') 100  200  300  400  500
3 [100, 200, 300, 400, 500] join('⭐️') 100⭐️200⭐️300⭐️400⭐️500
4 [100, 200, 300, 400, 500] join('-') 100-200-300-400-500

2. push( )

push( ) 메서드는 배열을 직접적으로 변경하는 메소드입니다. 하나 이상의 요소를 배열 마지막에 추가하며, 추가한 만큼 배열의 새로운 총 길이를 반환합니다. 

//02
const arrNum = [100, 200, 300, 400, 500];
const arrPush = arrNum.push(600); //마지막에 숫자 600 추가

console.log(arrPush); //리턴값 6
console.log(arrNum);  //결과값 100, 200, 300, 400, 500, 600
표로 확인해보자!
번호 기본값 메서드 리턴값 결과값
1 [100, 200, 300, 400, 500] push() 6 100,200,300,400,500,600

3. pop( )

pop( )은 배열을 직접적으로 변경하는 메소드입니다. 배열의 가장 마지막 요소를 삭제하고 삭제된 요소를 반환합니다. 때문에 pop() 메서드를 실행할 때 마다 배열의 길이가 1씩 줄어듭니다. 

//03
const arrNum = [100, 200, 300, 400, 500];
const arrPop = arrNum.pop( );   //마지막 요소(숫자) 삭제

console.log(arrPop); //리턴값 500(마지막 요소 불러옴)
console.log(arrNum);  //결과값 100, 200, 300, 400
표로 확인해보자!
번호 기본값 메서드 리턴값 결과값
1 [100, 200, 300, 400, 500] pop() 500 100,200,300,400
