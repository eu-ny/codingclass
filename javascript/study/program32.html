🖱 프로그래머스 1단계 : 최대공약수와 최소공배수 🖱

📖 문제 설명 📖

두 수를 입력받아 두 수의 최대공약수와 최소공배수를 반환하는 함수, solution을 완성해 보세요.
배열의 맨 앞에 최대공약수, 그다음 최소공배수를 넣어 반환하면 됩니다. 예를 들어 두 수 3, 12의 최대공약수는 3, 최소공배수는 12이므로 solution(3, 12)는 [3, 12]를 반환해야 합니다.

🧸 제한 사항 🧸

두 수는 1이상 1000000이하의 자연수입니다. 

function solution(n, m) {
    let a = 0;
    for(let i=0; i<=n; i++){
        if(n % i == 0 && m % i == 0){
            a = i
        }
    }
    let b = n * m / a
    let answer = [a,b]

    return answer
}
// 3과 12일 때  ==  2 * 2 * 3 = 12 최소 공배수 / 3 = 최대공약수

🧸 문제 풀이 🧸

1. 최대공약수 : 두 수의 공통인수를 곱한 것.
2. 최소공약수 : 두 수의 모은 인수를 곱한 것.
3. 반복문을 통해 첫번째 수만큼 반복시키고 조건문을 통하여 두 수 모두를 i로 나누었을 때 나누어 떨어진다면 변수 a에 저장한다.
4. 그 다음 두 수를 곱한 값에 a를 나누면 나오는 값을 변수 b에 저장한다. 이때 a는 최대공약수 b는 최소공배수가 된다.
