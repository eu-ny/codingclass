🖱 프로그래머스 1단계 : 가운데 글자 가져오기 🖱

📖 문제 설명 📖

단어 s의 가운데 글자를 반환하는 함수, solution을 만들어 보세요. 단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.

🧸 제한 사항 🧸

s는 길이가 1 이상, 100이하인 스트링입니다. 

function solution(s) {
    let string = s.length
    let answer = '';
        (string % 2 == 0) ? answer = s.substring((string/2)-1, (string/2)+1) : answer = s.substring((string/2), (string/2)+1)
    return answer;
}
            
//substring(int beginIndex, int endIndex) : 문자열에서 가져올 시작과 끝 부분을 정하여 자릅니다.
// s = "abcde" 일 때 "c"를 출력합니다. / s = "qwer"일 때 "we"를 반환합니다.

🧸 문제 풀이 🧸

1. 먼저 변수를 만들어서 문자열의 길이값을 저장합니다. (문자열에서 길이는 단어의 갯수이다.)
2. 조건삼항자를 통하여 문자열의 길이가 짝수일 경우에는 가운데 2글자를 가져오도록 합니다. 이때 substring이라는 문자열 메서드를 사용하여 문자열을 슬라이스 해줍니다. (문자열을 반으로 나눈후 각 -1 +1로 합니다.) 3. 만약 홀수일 경우 가운데 숫자를 가져오도록 합니다.

🧸 다른 사람의 풀이 🧸

function solution(s) {
    return s.substr(Math.ceil(s.length / 2) - 1, s.length % 2 === 0 ? 2 : 1);
}
