🖱 프로그래머스 1단계 : x만큼 간격이 있는 n개의 숫자 🖱

#1. 함수 solution은 정수 x와 자연수 n을 입력 받아, x부터 시작해 x씩 증가하는 숫자를 n개 지니는 리스트를 리턴해야 합니다. 다음 제한 조건을 보고, 조건을 만족하는 함수, solution을 완성해주세요.

제한 사항 : x는 -10000000 이상, 10000000 이하인 정수입니다.
n은 1000 이하인 자연수입니다. 

function solution(x, n) {
        var answer = [];
        let num = x;
        for(let i=0; i<n; i++){
            answer.push(x);
            x += num;
        }
        return answer;
    }

반복문을 통해 n번 만큼 X를 push 해주었습니다. 즉 4와 5를 대입햇을 경우 4씩 3번 더해져서 총 4, 8, 12 값이 push 됩니다~~!
