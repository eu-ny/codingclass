🖱 페럴렉스 효과 : 이질감 효과 🖱

페럴랙스(parallax)란 의미로 사용자가 스크롤링을 할 때 발생하는 이벤트(배경과 요소마다 속도 및 크기의 차이를 주어 입체적으로 느껴지게 하는 효과 등)를 말합니다.
이번 시간에는 스크롤을 내릴 때 넘버, 이미지, 텍스트의 각 위치값에 오차를 주어서 다르게 주어 이질감을 주는 효과를 실습해보겠습니다.🤩
추가로 필요한 소스는 GSAP로 스크롤 이벤트를 부드럽게 작동하기 위해서 입니다~! 그럼 실습하러 고고릥~ 🚗 💨

#2. JAVASCRIPT : 스크립트로 페럴랙스 효과 주기

먼저 전체적인 흐름을 보고 주석과 함께 스크립트가 어떻게 작동되는지 확인해봅시다! 🥹

오늘 스크립트에서도 지난 시간과 같이 재귀함수를 사용하여 스크립트를 완성하였습니다.
즉 오늘의 스크립트 포인트는! 재귀함수와 이질감을 주도록 이동 거리의 오차를 구하는 식, 그리고 gsap을 이용한 스크롤 에니메이션 구현 입니다~!
그럼 코드보러 슝~~ 해보아요~🚗💨 (재귀함귀에 대한 설명은 페럴렉스04번을 참고해주세요~)

💡 전체 스크립트 보기 
//함수 설정
function scroll(){

    //01 scrollTop값 구하고 값을 선택 요소에 출력
    let scrollTop = window.pageYOffset || document.documentElement.scrollTop;

    document.querySelector("#parallax__info span").innerText = Math.ceil(scrollTop);

    //02 이질감 효과 주기
    document.querySelectorAll(".content__item").forEach(item => {
        //선택자
        //선택한 콘텐츠(넘버, 이미지, 텍스트)들을 타겟(변수)에게 설정한다.
        const taget1 = item.querySelector(".content__item__img");
        const taget2 = item.querySelector(".content__item__desc");
        const taget3 = item.querySelector(".content__item__num");

        //이동시킬 거리 (오차주기)
        //현재 스크롤 탑 값이 1000일 경우 -item.offsetTop(섹션의 시작 스크롤값 위치)를 하면 0이 나온다. 이때 스크롤을 내리면 scrollTop은 다시 본래의 값으로 돌아오며 내렸을 때의 값에서 다시 item.offsetTop을 뺐을 때 나오는 값에 0.1을 곱할 경우 나오는 값이 변수에게 대입되는 것이다.
        //scrollTop과 item.offsetTop값은 같기 때문에 0이 나오는 것.
        //즉 scrollTop이 content__item(섹션)의 시작 위치 값에 도달했을 때 값을 0으로 만든 뒤 다시 scrollTop이 이동할 때 나오는 값에 원하는 오차값을 곱하여 반환되는 값을 구한다. 그 값이 y축의 이동 거리가 된다.
        //다음 content__item(섹션)에 도달할 경우 값은 다시 0이 된다.
        let offset1 = (scrollTop - item.offsetTop) * 0.1;
        let offset2 = (scrollTop - item.offsetTop) * 0.15;
        let offset3 = (scrollTop - item.offsetTop) * 0.2;
        
        //03 GSAP
        // 스크립트로 스타일을 설정할 경우 : css스타일 중 transform을 통해 이미지와 텍스트의 각 고정위치에서 움직이려는 거리를 설정한다. 각각이 콘텐츠들의 이동 거리가 다르므로 스크롤이 움직일 때 이질감을 준다.
        // taget1.style.transform = `translateY(${offset}px)`;
        // taget2.style.transform = `translateX(${offset}px)`;

        //gsap으로 스타일 주기 : 더 많은 애니메이션 속성 설정이 가능하며 움직임이 부드럽다. (한 번에 스타일을 선언할 수 있어 편리) : 주의할 점은 꼭 gsap의 CDN을 가져와야 한다는 것.
        //스타일을 줄 요소, {duration : 지속시간, translateX or Y(gsap에서는 x, y): 선택한 축 방향으로 이동할 값, ease: 전환 효과 (움직임 설정 : 시작과 끝의 속도 조절 등)}
        gsap.to(taget1, {duration: .3, y: offset1, ease: "power4.out"});
        gsap.to(taget2, {duration: .3, y: offset2});
        gsap.to(taget3, {duration: .3, y: offset2, ease: "expo.out"});
    });

    //04 : 1초에 60번 실행
    //함수가 무한하게 반복되어도 한 번 움직일 때 60번만 실행이라는 제한이 걸려있기 때문에 서버가 다운이 되지 않는다.
    requestAnimationFrame(scroll); 
}
scroll(); //실행문

핵심 자바스크립트 📌

document.querySelectorAll(".content__item").forEach(item => {
    let offset1 = (scrollTop - item.offsetTop) * 0.1;   //100일 경우 10
    let offset2 = (scrollTop - item.offsetTop) * 0.15;  //100일 경우 15
    let offset3 = (scrollTop - item.offsetTop) * 0.2;   //100일 경우 20
});

1. scrollTop이 item.offsetTop(= content__item.offsetTop)의 시작 위치 값에 도달했을 때 값을 0으로 만든 뒤 다시 scrollTop이 이동할 때 나오는 값에 원하는 오차값을 곱하여 반환되는 값을 변수에 대입합니다.
2. 변수에 대입 된 값이 바로 다음에 설정할 translateY의 이동 거리가 됩니다. 즉 0에서 100만큼 이동한다 가정하여 곱한 값이 0.1일 때 Y축으로 10px을 움직이게 되는 것입니다.
3. 그 뒤 다음 item.offsetTop(= content__item)에 도달할 경우 값은 다시 0이 됩니다. 

document.querySelectorAll(".content__item").forEach(item => {
    gsap.to(taget1, {duration: .3, y: offset1, ease: "power4.out"});
    gsap.to(taget2, {duration: .3, y: offset2});
    gsap.to(taget3, {duration: .3, y: offset2, ease: "expo.out"});
});

1. gsap을 사용하여 CSS 움직임 설정을 해줄 경우 기본 스크립트로 구현하는 것보다 더 많은 속성과 부드러운 움직임을 설정할 수 있게 됩니다.
2. 사용 방법은 스타일을 줄 요소, {duration : 지속시간, translateX or Y(gsap에서는 x, y): 축 방향으로 이동할 값, ease: 전환 효과 (움직임 설정 : 시작과 끝의 속도 조절 등)}으로 작성하시면 됩니다.
3. 이 때 translateY에 offset1을 대입했기 때문에 만약 전체 100px을 움직일 경우 translateY는 10px을 움직이게 됩니다.

TIP! 좀 더 다양한 ease를 설정하고 싶은 경우 GSAP 홈페이지에 들어가서 확인하시면 됩니다! 또한 GSAP을 사용할 경우 반드시 CDN을 가져와야 합니다!!

📖 정리 📖

1. 섹션1 offsetTop값 : 500px / 섹션2 offsetTop값 : 1000px => scrollTop == 500px / scrollTop == 1000px 일 때 0px이 됩니다.

2. 다시 움직인 scrollTop값이 700일 때 다시 섹션1의 offsetTop값을 뻰 경우 200이 되며 * 0.1을 했을 때 20을 반환하게 됩니다. 즉 최종적으로 translateY에게 이 값을 대입하게 되면 스크롤을 내릴 때 해당 컨텐츠가 20px씩 내려가게 됩니다. 이때 이동 효과를 더 크게 하고 싶은 경우 곱하는 값을 더 크게 설정하시면 됩니다.

3. 그 뒤 다음 섹션 값인 1000에 되었을 때 scrollTop값은 다시 0이 됩니다. 때문에 계속해서 모든 content__item들의 애니메이션이 작동되는 것 입니다.
