🖱 마우스 효과 : 이미지효과 🖱

이번 시간에는 이미지 영역안에 커서가 들어갈 경우 스타일이 바뀜과 동시에 이미지도 커서 반대 방향으로 움직이는 효과를 주도록 해보겠습니다.

#1. HTML / CSS 설정하기

! HTML TIP !
figure는 이미지나 그래프 등의 도표를 문서에 삽입할 경우 사용하는 시멘틱 태그입니다. figure를 통해 이미지의 영역을 잡아줍니다!
figcaption은 figure의 설명을 도와주는 시멘틱 태그라 볼 수 있습니다. figcaption을 통해 텍스트의 영역을 잡아줍니다~!

✏️ HTML 스크립트 보기 
<main id="main">
    <section id="mouseType04">
        <div class="mouse__cursor"></div>
        <div class="mouse__wrap">
            <figure>
                <img src="../assets/img/Mountian_bg03.jpg" alt="image">
                <figcaption>
                    <p>The way get started is to quit talking and begin doing.</p>
                    <p>무언가를 시작하려면 말하는 것을 멈추고 행동으로 옮겨야 한다.</p>
                </figcaption>
            </figure>
        </div>
    </section>
  
    <div class="mouse__info">
        <ul>
            <li>mousePageX : <span class="mousePageX">0</span>px</li>
            <li>mousePageY : <span class="mousePageY">0</span>px</li>
            <li>centerPageX: <span class="centerPageX">0</span>px</li>
            <li>centerPageY: <span class="centerPageY">0</span>px</li>>
        </ul>
    </div>
</main>

! CSS TIP !
position: absolute; : 기존의 마우스 커서를 따라다니도록 포지션을 정해줍니다. (스크립트와 GSAP을 통해 left와 top값이 이동할 때 마다 변하도록 설정해줄 것이므로 왼쪽 상단 모서리에 잠시 위치시켜 줍니다.)
object-fit: cover는 background-size: cover;와 같은 속성으로 백그라운드 설정이 아닌 img태그에게 직접 사용할 수 있다.

#2. JAVASCRIPT : 스크립트 짜기

먼저 getBoundingClientRect()을 통해 요소(커서)의 모든 정보값을 불러온 뒤 이벤트 메서드인 mousemove()와 GSAP을 통해 스크립트를 완성해보자!
오늘의 포인트는 바로 브라우저의 전체 너비와 길이의 반을 구하고 커서 영역의 너비와 길이의 반을 구하는 것 입니다~!
그럼 아래의 전체 스크립트와 주석으로 흐름을 파악한 뒤 핵심 스크립트로 다시 한 번 이해해봅시다.

💡 전체 스크립트 보기 
//01. 선택자
const cursor = document.querySelector(".mouse__cursor"); //커서
const cursorRect = cursor.getBoundingClientRect();       //커서위치값

//02. 마우스를 움직일 때 이벤트 설정 (이미지를 대상으로 이벤트 메서드를 사용했으므로 이미지 영역 내에서 마우스 이벤트 발생)
document.querySelector(".mouse__wrap figure").addEventListener("mousemove", (e) => {
    //03. 커서
    //커서의 영역을 만들게 될 경우 현재 커서의 포지션이 left와 top값을 기준으로 설정되어 있기 때문에 사각형의 왼쪽 모서리를 기준으로 영역이 생긴다. 
    //즉 왼쪽모서리 아래쪽으로 영역이 생기기 때문에 거리감이 생기겍 된다 특히 원형으로 영역을 잡을 경우 사각형이 깍여져서 만들어지기 때문에 더욱 거리감이 생기게된다. 
    //때문에 사각형(커서 영역)을 ÷2 해주어서 커서를 가운데로 지정해주는 것! : e.pageX - cursorRect.width/2 : 요소의 x축 - 요소의 영역 가로 길이의 반
    gsap.to(cursor, {
        duration: .5,
        left: e.pageX - cursorRect.width/2,
        top: e.pageY - cursorRect.height/2,
    });

    //04. 마우스 좌표 값
    //요소의 x축과 y축 좌표값을 변수에 대입한다.
    let mousePageX = e.pageX;
    let mousePageY = e.pageY;

    //전체가로
    //innerWidth || outerWidth || screen.Width
    // window.innerWidth //1440 : 브라우저 기준 크기 (스크롤바 미포함)
    // window.outerWidth  //1920 : 브라우저 기준 크기 (스크롤바 포함)
    // window.screen.Width  //1920 : 디바이스 기준(모니터 등등의 화면 크기로 계산)
    // console.log(window.innerWidth);

    //05. 마우스 좌표값 가운데로 초기화 : 변수에 초기화 된 값을 저장해준다.
    //전체길이/2(전체길이 ÷ 2) - 마우스 좌표값 == 0 : 500*500일 경우 전체 길이에서 반을 빼주면 가운데로 맞춰지게 된다.
    //전체 영역을 가운데로 초기화 == window.innerWidth/Height/2 -> 그 뒤 각 마우스의 좌표값을 빼주면 좌표값 기준도 가운데로 초기화 된다. 즉 이 식의 최종 값이 바로 아래의 이미지의 이동 거리 값이 된다. (커서가 중앙에 위치 == 0(가운데로 초기화) | 커서가 왼쪽에 위치 == 양수 | 커서가 오른쪽에 위치 - 값(음수))
    //크기 속성을 사용할 땐 앞에 윈도우는 생략해도 괜찮다.
    let centerPageX = window.innerWidth/2 - mousePageX;
    let centerPageY = window.innerHeight/2 - mousePageY;

    //06. 이미지 움직이기
    const imgMove = document.querySelector(".mouse__wrap figure img");
    //기존 js 방식
    // imgMove.style.transform = "translate(" + centerPageX/20 + "px, " + centerPageY/20 + "px)";

    //GSAP
    //좌표를 20으로 나눈 이유 : 오차 값으로 이미지가 이동 할 경우 너무 많이 이동하기 때문에 20을 나눠주어 이미지의 이동거리를 축소시켜 준 것.
    gsap.to(imgMove, {
        duration: 0.3,
        x: centerPageX/20,
        y: centerPageY/20
    });

    //07. 마우스 좌표값 출력
    //textContent를 통해 선택자에게 각 좌표값를 대입시킨다.
    document.querySelector(".mousePageX").textContent = mousePageX;
    document.querySelector(".mousePageY").textContent = mousePageY;
    document.querySelector(".centerPageX").textContent = centerPageX;
    document.querySelector(".centerPageY").textContent = centerPageY;
});

✏️ 스크립트 보기 : 자바스크립트

//01 선택자
const cursor = document.querySelector(".mouse__cursor"); //커서
const cursorRect = cursor.getBoundingClientRect();       //커서위치값

//02 마우스를 움직일 때 이벤트 설정
document.querySelector(".mouse__wrap figure").addEventListener("mousemove", (e) => {

    //03 커서 : GSAP
    gsap.to(cursor, {
        duration: .5,
        left: e.pageX - cursorRect.width/2,
        top: e.pageY - cursorRect.height/2,
    });

    //04 마우스 커서 좌표 값
    let mousePageX = e.pageX;
    let mousePageY = e.pageY;


    //05 마우스 커서 좌표값 가운데로 초기화 후 변수에 저장하기
    //전체길이/2(전체길이 ÷ 2) - 마우스 커서 좌표값 == 0
    let centerPageX = window.innerWidth/2 - mousePageX;
    let centerPageY = window.innerHeight/2 - mousePageY;

    //06 이미지 움직이기
    const imgMove = document.querySelector(".mouse__wrap figure img"); //이미지

    //GSAP : 20을 나눠주어 이미지의 이동거리를 축소시켜 준 것.
    gsap.to(imgMove, {
        duration: 0.3,
        x: centerPageX/20,
        y: centerPageY/20
    });

    //07 마우스 좌표값 출력
    document.querySelector(".mousePageX").textContent = mousePageX;
    document.querySelector(".mousePageY").textContent = mousePageY;
    document.querySelector(".centerPageX").textContent = centerPageX;
    document.querySelector(".centerPageY").textContent = centerPageY;
});

! JAVASCRIPT TIP !
1. 커서와 커서의 위치, 크기 정보값을 모두 불러오도록 선택자를 만들어 줍니다.

2. 이미지를 타겟으로 하여 이벤트 설정을 해줍니다.

3. 첫번째로 GSAP을 통해 커서의 움직임을 설정해주고 커서의 위치를 커서 영역(CSS속성으로 정해준 값)의 가운데로 지정해줍니다. : 너비와 길이의 ÷ 2를 해주면 중앙으로 맞춰진다.

4. 변수를 만들어서 마우스 커서의 좌표값을 저장해줍니다. -> 이 변수는 이미지를 움직이기 위해 마우스 커서의 좌표값을 가운데로 초기화 해주는 식에 사용됩니다~!

5. 먼저 브라우저의 전체 너비와 길이의 반을 구한뒤 마우스 커서 좌표값을 빼주어서 가운데 좌표값 0으로 초기화 되도록 합니다.

6. 이미지를 움직일 수 있도록 선택자를 만들어 준 뒤 GSAP을 이용하여 X / Y 축 값에 위의 변수 ÷ 20을 해주어 이미지의 이동 좌표를 설정해줍니다.

7. 마지막으로 마우스 인포에 마우스의 각 좌표값이 출력될 수 있도록 선택자와 textContent를 통해 좌표속성 값을 출력시킵니다.
