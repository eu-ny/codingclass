데이터 타입

데이터 타입은 변수에 저장되는 데이터의 유형으로 원시 데이터 타입과 객체 데이터 타입으로 나눌 수 있다.

#1. 숫자(number) 데이터

숫자 데이터는 정수, 소수점 등을 표현 할 수 있다.

let table = 1;
let table2 = 1.5;

#2. 문자(string) 데이터

문자 데이터는 '' 또는 "" 로 표현할 수 있다. 보통 문자 데이터를 사용할 땐 작은 따옴표 '' f를 많이 표현
문자안에 문자가 들어갈 땐 서로 다른 따옴표를 사용해주어야 한다.

let table = "문자와 '문자"
let table2 = '문자와 "문자"'

#3. 논리(boolean) 데이터

논리 데이터는 true와 false의 값을 표현한다.
논리 데이터에서 0값은 false를 의미하며 0값 이외의 숫자 및 문자 값은 true를 의미한다.
Boolean()명령어 = 변수의 값을 true나 false로 변환

let table = (5 > 2); // true
let table2 = (5 < 2); // false
let table3 = 0; // false
let table3 = 1; // true

#4. 배열( [] ) 데이터

배열 데이터는 순서가 있는 데이터 집합이며, 변수 하나에 여러개의 값을 순차적으로 저장시킬 수 있다.

let table = ["자바스크립트","100"];
let table2 = [100, 200];

#5. 객체( {} ) 데이터

다수의 property 또는 method를 하나로 묶어놓은 형태를 객체 데이터라 칭하며 데이터 형태는 키와 값으로 되어있다. 즉 하나의 데이터 덩어리라 볼 수 있다.
객체에는 함수, 배열, 문자열이 들어갈 수 있다.

let table = {a:100, b:200};
let table2 = {};

#5. 특수값 데이터

null과 undefined의 공통점은 논리 데이터에서 둘 다 false로 표현된다.

undefined 데이터

변수를 선언하고 데이터 값은 지정하지 않았을 때 결과 값으로 나타나는 데이터이다.
객체의 속성값을 지정하지 않았을 경우에 자동으로 저장되는 값이기도 하다.

let table = 10;
let table2; // undefined

const obj = {};
obj.table = 10;
obj.table2; //undefined

null 데이터

변수를 빈 상태로 초기화 하거나 데이터를 저장했지만 값이 존재하지 않을 때 null값을 반환한다.

var obj = 10;
obj = null;
console.log(obj); // null 변수를 빈 상태로 만든다.
