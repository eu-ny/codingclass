🖱 프로그래머스 1단계 : 7의 개수 구하기 🖱

📖 문제 설명 📖

두 수를 입력받아 두 수의 최대공약수와 최소공배수를 반환하는 함수, solution을 완성해 보세요.
배열의 맨 앞에 최대공약수, 그다음 최소공배수를 넣어 반환하면 됩니다. 예를 들어 두 수 3, 12의 최대공약수는 3, 최소공배수는 12이므로 solution(3, 12)는 [3, 12]를 반환해야 합니다.

🧸 제한 사항 🧸

두 수는 1이상 1000000이하의 자연수입니다. 

function solution(array) {
    let arr = array.join("").split("")
    return arr.filter(el => el === '7').length;
}
// filter() :

🧸 문제 풀이 🧸

1. join("").split("")를 통해 배열 내 정수를 하나씩 쪼개준다.
2. filter()를 사용하여 만약 그 값이 문자열 7일 경우 그 갯수를 반환하도록 한다.

🧸 다른 사람의 풀이 🧸

function solution(array) {
    return array.join('').split('7').length-1;
}

split으로 7을 바로 뺄 수 있네요..!
