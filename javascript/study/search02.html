includes()를 통해 검색 사이트 제작하기

이번 시간에는 문자열 메서드 중 하나인 includes() 메서드를 통해 검색 사이트를 제작해보겠습니다!
먼저 includes() 메서드에 대해 짧게 알아보고 사이트 실습에 들어가보겠습니다!

#1. includes()

includes() 메서드는 대소문자를 구분하며 문자열 내에서 검색할 문자의 포함 여부에 따라 불린(true, false)을 반환합니다.

✏️예제 코드

const str1 = "javascript reference";

const currentStr1 = str1.includes("javascript");      // true
const currentStr2= str1.includes("j");                // true
const currentStr3= str1.includes("b");                // false : b는 포함되어 있지 않으므로 false
const currentStr4= str1.includes("reference");        // true
const currentStr5 = str1.includes("reference", 1);    // true
const currentStr6 = str1.includes("reference", 11);   // true
const currentStr7 = str1.includes("reference", 12);   // false

#4. JAVASCRIPT : 스크립트 짜기

includes()와 반복문, 조건문을 통해 스크립트를 완성해보자!
코드의 주석과 함께 스크립트를 이해해보도록 합니다:)

✏️스크립트 보기

//선택자
const searchBox = document.querySelector(".search__box input");       //검색 영역
const searchList = document.querySelectorAll(".search__list ul li");  //목록 리스트
const searchInfo = document.querySelector(".search__info .num");      //갯수

//전체 갯수 구하기
searchInfo.textContent = searchList.length;

//searchBox 검색 영역
searchBox.addEventListener("keyup",()=>{
    const searchWord = searchBox.value;     // 사용자가 입력한 키워드

    searchList.forEach(el => {
        const cssName = el.dataset.name; //css속성 데이터가 들어간 요소를 모두 가져옴 = CSS 속성의 모든 갯수값 출력
        const cssType = el.dataset.type; //CSS 속성의 타입별 갯수값을 가져옴

        if(cssName.includes(searchWord) | cssType.includes(searchWord)){  //사용자가 입력한 값이 true일 때 show false면 숨기기
            el.classList.remove("hide");
        } else {
            el.classList.add("hide");
        }
    });
});

! 검색 영역이 작동되는 과정 알아보기 !
***상수명.includes(검색값) 메서드는 사용자가 검색한 문자열이 상수에 저장한 데이터 내에 포함되었을 때 true를 반환하게 됩니다.***
1. 이벤트리스너를 통해 keyup(= 한 글자라도 입력하면 실행)을 실행시킵니다.
2. classList를 통해 hide라는 클래스를 추가해주어 사용자가 입력한 값이 포함되어 있지 않은 경우 false가 반환되어 속성 리스트를 숨깁니다.(→ css 속성값으로 hide에게 display: none;을 주었기 때문!)
3. 값이 true일 경우 remove를 실행하여 추가했던 hide를 지우고 속성 리스트를 보이게 합니다.

✏️ 쫌쫌따리 미션 : for()문과 forEach()문을 사용하여 전체 갯수 구하는 식 만들기!

그동안 자주 나왔던 반복문들을 이용해서 전체 갯수를 구하는 식도 만들어 봅시다! 

for(let i=0; i<=searchList.length; i++){
    document.querySelector(".search__info .num").innerText = i;
}

searchList.forEach((el, i) => {
        document.querySelector(".search__info .num").innerText = i + 1;
});
