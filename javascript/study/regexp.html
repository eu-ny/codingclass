자바스크립트 : 정규 표현식(패턴)

정규 표현식이란? 특정한 규칙을 가진 문자열의 집합을 표현하는 데 사용하는 형식 언어이며 문자열에서 특정 문자 조합을 찾기 위한 패턴입니다!
즉 쉽게 말해서 문자열에서 특정 문자를 찾아내는 도구라고도 볼 수 있습니다!

#1. 정규 표현식 객체 주요 패턴

정규 표현식에서 사용하는 기호를 메타(meta) 문자라고 합니다.

패턴 설명
abc abc 문자열을 검색합니다.
/abc/는 'abc'
[abc] a, b, c 중 문자 하나를 검색합니다.
/[abc]d/는 'ad', 'bd', 'cd'
[^abc] a, b, c 중 문자 하나를 검색합니다.
/[abc]d/는 'ad', 'bd', 'cd'
[A-Z] 알파벳 대문자를 검색합니다.
[a-z] 알파벳 소문자를 검색합니다.
[0-9] 0부터 9까지의 숫자를 검색합니다.
. 하나의 문자를 검색합니다.
만약 '마침표' 문자 그대로의 의미를 사용할 경우 '₩.'으로 표현해야 합니다.
\w 알파벳, 숫자, _를 검색합니다.
\W 알파벳, 숫자, _를 제외하고 검색합니다.
\d 숫자를 검색합니다.
\D 숫자를 제외하고 검색합니다.
\s 하나의 공백을 검색합니다.
\S 공백을 제외하고 검색합니다.
^ 행의 첫 문자가 일치해야 함을 의미합니다.
$ 행의 끝 문자가 일치해야 함을 의미합니다.
* * 앞의 문자가 0번 이상 반복하는 것을 의미합니다.
/[a*b]/는 'b', 'ab', 'aab', 'aaab', 'aaa...b'
+ + 앞의 문자가 1번 이상 반복하는 것을 의미합니다.
/[a+b]/는 'ab', 'aab', 'aaab', 'aaa...b'
? ? 앞의 문자가 0번 또는 1을 의미합니다.
/[a?b]/는 'b', 'ab'
{} /₩d{3}/은 숫자 3개를 의미합니디.
/₩d{3, 5}/은 숫자 3 ~ 5개를 의미합니디.
/₩d{3,}/은 숫자 3개 이상을 의미합니디.
a|b a or b를 의미합니다.
(abc) 그룹화를 의미합니다.

! point ! 문자열에서 추출하고 싶은 문자와 메타 문자를 합친 것이 패턴이라고 볼 수 있으며 추출하고 싶은 문자에 맞게 패턴을 사용하면 된다.

효율성의 증가!
자칫 길어지고 복잡해질 수 있는 코드들을 짧고 간결하게 작성할 수 있어 효율적으로 처리할 수 있게 되기 때문입니다. 코딩에서는 길고 복잡할 수록 좋지 않은 코드이기 때문에 효율성을 중요하게 여겨야 합니다.
예시
1.문자열을 검열하기 위한 규칙 설정하기(사용자 입력 유효성 검증(이메일, 주민등록번호, 비밀번호 등등)) : 조건문으로 작성할 시 하나하나의 케이스를 조건으로 작성해야 하기 때문에 코드가 매우 길고 복잡해진다.
