🖱 프로그래머스 1단계 : 3진법 뒤집기 🖱

📖 문제 설명 📖

자연수 n이 매개변수로 주어집니다. n을 3진법 상에서 앞뒤로 뒤집은 후, 이를 다시 10진법으로 표현한 수를 return 하도록 solution 함수를 완성해주세요.

🧸 제한 사항 🧸

n은 1 이상 100,000,000 이하인 자연수입니다. 

function solution(n) {
    var answer = n.toString(3).split('').reverse().join('');
    return parseInt(answer, 3);
}
// n이 45일 때 7이 return 됩니다.

🧸 문제 풀이 🧸

1. toString으로 수를 문자열로 변환시킨 뒤 split('').reverse().join('')으로 하나씩 쪼갠 후 리버스해주었습니다. 2. 그 후 parseInt를 사용하여 문자열을 다시 숫자로 변환시켜 주었습니다.

🧸 다른 사람의 풀이 🧸

function solution(n) {
    return Number.parseInt(n.toString(3).split("").reverse().join(""),3)
}

Number와 parseInt()으로 식을 묶어서 바로 리턴되도록 했네요!
