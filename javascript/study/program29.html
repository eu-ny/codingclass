🖱 프로그래머스 1단계 : 행렬의 덧셈 🖱

📖 문제 설명 📖

행렬의 덧셈은 행과 열의 크기가 같은 두 행렬의 같은 행, 같은 열의 값을 서로 더한 결과가 됩니다. 2개의 행렬 arr1과 arr2를 입력받아, 행렬 덧셈의 결과를 반환하는 함수, solution을 완성해주세요.

🧸 제한 사항 🧸

행렬 arr1, arr2의 행과 열의 길이는 500을 넘지 않습니다. 

function solution(arr1, arr2) {
    let answer = [];
    
    for(i=0; i<arr1.length; i++){
        let num = [];
        for(j=0; j<arr1[i].length; j++){
            num.push(arr1[i][j] + arr2[i][j])
        }
        answer.push(num)
    }
    return answer;
}
// push()는 배열의 끝에 하나 이상의 요소를 추가하고, 배열의 새로운 길이를 반환합니다.
// arr1 == [[1,2],[2,3]] arr2 == [[3,4],[5,6]]일 때 [[4,6],[7,9]]를 반환합니다.
// arr1 == [[1],[2]] arr2 == [[3],[4]]일 때 [[4],[6]]를 반환합니다.

🧸 문제 풀이 🧸

1. 먼저 변수에 빈 배열을 만들어 줍니다.
2. 반복문을 통해 arr1의 길이 만큼 반복해줍니다. 그 뒤 배열안에 또 배열을 만들어주기 위해 빈 배열을 저장한 변수를 하나 만들어줍니다.
3. num안에 push()를 사용하여 arr1의 i번째 배열안의 j번째 배열의 값 + arr2의 i번째 배열안의 j번째 배열의 값을 넣어 줍니다.
4. 그 뒤 num에 저장된 2개의 배열을 처음에 만들어 둔 빈 배열에 push하여 두개의 배열을 저장합니다.
