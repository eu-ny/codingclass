🖱 프로그래머스 1단계 : 없는 숫자 더하기 🖱

📖 문제 설명 📖

0부터 9까지의 숫자 중 일부가 들어있는 정수 배열 numbers가 매개변수로 주어집니다. numbers에서 찾을 수 없는 0부터 9까지의 숫자를 모두 찾아 더한 수를 return 하도록 solution 함수를 완성해주세요.

🧸 제한 사항 🧸

1 ≤ numbers의 길이 ≤ 9
0 ≤ numbers의 모든 원소 ≤ 9
numbers의 모든 원소는 서로 다릅니다. 

function solution(numbers) {
    let answer = 0;
    
    for(let i=0; i < 10; i++) {
        if(!numbers.includes(i)) answer += i;
    }
    return answer;
}

// 조건문에서 !가 있는 경우 : 해당 조건이 아닌 경우 조건문을 실행시킨다.
// if(!numbers.includes(i)) answer += i은 numbers에 i가 includes(포함여부를 알려주는 메서드)되지 않았을 경우 answer에 i를 저장시킨다.
// numbers = [1,2,3,4,6,7,8,0] 일 때 14를 반환합니다.

🧸 문제 풀이 🧸

1. 반복문을 통해 0부터 9까지 반복시킵니다.
2. 조건문을 통해 만약 numbers의 배열에 i가 포함되어있지 않은 경우 i를 answer 변수에 저장시킵니다.
3. +=이므로 저장된 i값의 총합을 구하여 그 값을 리턴합니다.
