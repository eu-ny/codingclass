🖱 마우스 효과 : 마우스 포인터 따라다니기 🖱

이번 시간에는 마우스 커서를 움직임에 따라 해당 영역에 커서가 닿았을 때 이벤트가 발생하는 효과를 실습해보겠습니다 😆

#1. HTML / CSS 설정하기

! HTML TIP !
페이지 섹션과 마우스 인포(커서 위치값 박스)를 각각 나누어 구조에 맞게 레이아웃을 짠 뒤 스크립트를 위한 선택자를 각각 부여합니다.
또한 텍스트에서 스타일 변경 및 이벤트를 발생시키고 싶은 곳에 각각 선택자를 선언합니다.

<section id="mouseType01">

#2. JAVASCRIPT : 스크립트 짜기

이벤트 메서드인 mouseover()과 mouseout() 그리고 위치 속성들을 통해 스크립트를 완성해보자!

💡 전체 스크립트 보기 
//커서 위치 좌표 값
window.addEventListener("mousemove", (event) => {
    document.querySelector(".clientX").innerText = event.clientX; //브라우저 기준
    document.querySelector(".clientY").innerText = event.clientY;
    document.querySelector(".offsetX").innerText = event.offsetX; //요소 기준
    document.querySelector(".offsetY").innerText = event.offsetY;
    document.querySelector(".pageX").innerText = event.pageX; //문서 기준
    document.querySelector(".pageY").innerText = event.pageY;
    document.querySelector(".screenX").innerText = event.screenX; //디바이스 기준
    document.querySelector(".screenY").innerText = event.screenY;
});

const cursor = document.querySelector(".mouse__cursor"); //지역 변수(함수안)는 이벤트가 작동 X 전역변수로 해야 이벤트가 작동되기 때문에 전역 변수로 작업.

//커서 영역 정하기

    cursor.style.left = e.clientX -25 + "px";
    cursor.style.top = e.clientY -25  + "px";
});

✏️ 스크립트 보기 : 자바스크립트

! JAVASCRIPT TIP !
1. 위치(좌표)값을 출력하기 위해 innerText를 통해 이벤트 위치 속성을 부여합니다.
2. 커서를 커서 영역의 중심에 맞추기 위해 브라우저 기준의 위치 속성인 clientX/Y를 사용합니다.
3. 이 때 반드시 문자열을 추가하여 그 값의 단위를 설정해주어야 합니다!(+ "px")
4. getAttribute 메서드를 사용하여 마우스 이벤트 메서드를 효율적으로 실행시킵니다.
5. 그 외에는 아래의 주석을 참고하여 더 자세하게 알아봅시다!
