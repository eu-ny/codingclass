🖱 프로그래머스 1단계 : 문자열을 정수로 바꾸기 🖱

#1. 문자열 s를 숫자로 변환한 결과를 반환하는 함수, solution을 완성하세요.

제한 사항 : s의 길이는 1 이상 5이하입니다.
s의 맨앞에는 부호(+, -)가 올 수 있습니다.
s는 부호와 숫자로만 이루어져있습니다.
s는 "0"으로 시작하지 않습니다. 

function solution(s) {
    return Number(s)
}

혹시 하고 메서드를 이용하여 숫자로 바꾸어주었는데 맞아서 놀랐습니다..! 문자열을 숫자로 바꿔주는 Number() 메서드를 통해 완료하였습니다.
