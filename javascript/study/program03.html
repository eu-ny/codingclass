🖱 프로그래머스 1단계 : 약수의 합 🖱

#1.정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

제한 사항 : n은 0 이상 3000이하인 정수입니다. 

function solution(n) {
    var answer = 0;
    for( i=1; i<=n; i++){
        if(n%i==0)
          answer+=i
    }
    return answer;
}

n이 12이라고 가정한 후 만약 n을 나누었을 때 1부터 12까지의 수 중 나머지가 0이 되는(딱 떨어지는 값)을 구하여 약수를 구해준 후 총합을 구해줍니다.
