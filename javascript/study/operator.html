자바스크립트 연산자

프로그램에 필요한 수식을 만들 떄 필요한 것이 연산자이다. 연산자의 종류는 산술연산자, 대입연산자, 비교연산자, 논리연산자 등이 있다.

#1. 산술 연산자

산술 연산자는 사칙연산을 다루는 기본적인 연산자이다. 그만큼 가장 많이 사용되기도 한다.

산술연산자 종류 설명
+ 더하기 연산을 할 때 사용
문자와 문자, 문자와 변수 등을 연결할 때 사용
- 빼기 연산을 할 때 사용
* 곱하기 연산을 할 떄 사용
/ 몫을 구할 때 사용
% 나머지를 구할 때 사용
++ 1씩 증가시킬 때 사용
-- 1씩 감소시킬 때 사용

보통 왼쪽의 피연산자에서 오른쪽 피연산자로 결합한다.

#2. 대입 연산자

대입 연산자는 변수에 값을 대입할 때 사용하는 연산자이다.
대입 연산자에서 '='는 같다의 의미가 아닌 변수에 '저장(대입)'을 의미

대입연산자 종류 설명
= num = 1은 우변의 값을 좌변의 num 변수에 대입(저장)
+= num += 2 는 num + 2와 같다.
-= num -= 2 는 num - 2와 같다.
*= num *= 2 는 num * 2와 같다.
/= num /= 2 는 num / 2와 같다.
%= num %= 2 는 num % 2와 같다.

보통 오른쪽에서 왼쪽으로 피연산자들을 결합한다.

#3. 비교 연산자

비교 연산자는 값을 비교하여 결과 값을 논리값인 'true'나 'false'로 반환한다.

비교 연산자 종류 설명
> a > b 가 참이면 true, 거짓이면 false
< a < b 가 참이면 true, 거짓이면 false
>= a >= b 가 참이면 true, 거짓이면 false
<= a <= b 가 참이면 true, 거짓이면 false
== a == b 가 참이면 true, 거짓이면 false
== 연산자의 경우 데이터 타입은 비교 X
ex) 5 == 5 는 false
!= a != b, a와 b가 같지 않으면 true, 같으면 false
=== a === b, a와 b가 같고 데이터 타입도 같으면 true, 같지 않으면 false
ex) 5 === 5 는 false
!== a !== b, a와 b가 같지 않거나 데이터 타입이 같지 않으면 true, 같으면 false

#4. 논리 연산자

논리 연산자는 주어진 논리식을 판단하여, 참(true)과 거짓(false)을 결정하는 연산자이다.

논리 연산자 종류 설명
&&
(AND연산자)		 a && b, a와 b 모두 true일 때 true
→논리식이 모두 참이면 참을 반환함
||
(OR연산자) 		a || b, a와 b 중 하나만 true일 때 true
→논리식 중에서 하나라도 참이면 참을 반환함
!
(NOT연산자) 		!a, a가 true일 때 false, false일 때 true로 바꾼다.
→논리식의 결과가 참이면 거짓을, 거짓이면 참을 반환함
