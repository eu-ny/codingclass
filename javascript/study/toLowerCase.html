자바스크립트 : 문자열 메서드 : 대소문자/공백

지금부터 자바스크립트의 대소문자 변경, 공백 제거 문자열 메소드에 대해 알아봅시다!

#1. toUpperCase(), toLowerCase()

toUpperCase()는 문자열을 대문자로 변경해주며 toLowerCase()는 소문자로 변경해줍니다. 변경 후에는 새로운 문자열로 반환해줍니다. 

//01. 대문자 변경
const str1 = "javascript";
const currentStr1 = str1.toUpperCase();

//02. 소문자 변경
const str2 = "JAVASCRIPT";
const currentStr2 = str1.toLowerCase();
표로 확인해보자!
번호 기본값 메서드 리턴값
1 javascript toUpperCase() JAVASCRIPT
2 JAVASCRIPT toLowerCase() javascript

✨ 응용 코드로 GO ✨
퀴즈 만들기 02에서도 위의 메서드를 이용하여 사용자 입력값을 변경시켜줍니다.

#2. trim() , trimStart(), trimEnd()

trim()은 문자열의 앞뒤 공백을 제거해주며 새로운 문자열로 반환시켜 줍니다.
trimStart(), trimEnd()도 마찬가지로 trimStart()는 앞의 공백을 trimEnd()은 뒤의 공백을 제거해줍니다. 

//01. trim()
const str1 = "       javascript        ";
const currentStr1 = str1.trim();
    

//02. trimStart()
const str2 = "       javascript        ";
const currentStr2 = str2.trimStart();


//03. trimEnd()
const str3 = "       javascript        ";
const currentStr3 = str3.trimEnd();
표로 확인해보자!
