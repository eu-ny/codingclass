jQuery : 기본 선택자

이번 시간에는 자바스크립트의 jQuery 기본 선택자에 대해 알아보겠습니다.😆

#1. jQuery 기본 선택자

jquery에서는 html요소와 관련된 기능 구현이 많기 떄문에 CSS와 밀접한 관계가 있습니다.
다음의 표와 함께 기본 선택자의 종류를 알아봅시다! 😇

선택자 종류 설명
태그 선택자 $("p") p 태그를 선택합니다.
id 선택자 $("#section") id가 section인 요소를 선택합니다.
class 선택자 $(".section") class가 section인 요소를 선택합니다.
자식 선택자 $("#section > ul > li") #section의 자식 요소 중 li를 선택합니다.
인접 선택자 $("#section + #content") #section의 바로 다음에 오는 #content를 선택합니다.
형제 선택자 $("#section ~ #content") #section의 형제 요소인 #content를 선택합니다.
종속 선택자 $("div.logo") div 태그 중 클래스가 logo인 요소를 선택합니다.
그룹 선택자 $("#section .content .logo") #section과 .content .logo 요소들을 모두 선택합니다.
전체 선택자 $("*") 모든 요소를 선택합니다.
