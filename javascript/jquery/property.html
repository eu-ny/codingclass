Jquery : 속성 선택자

이번 시간에는 자바스크립트의 jquery 속성 선택자에 대해 알아보겠습니다.😆

#1. jquery 속성 선택자

다음은 속성 선택자의 종류 입니다. 표와 예제를 보고 이해해봅시다!

선택자 종류 설명
요소[속성] $("p[class]") p 요소 중 class 속성을 가지고 있는 요소를 선택합니다.
요소[속성='값'] $("p[class='abc']") p 요소 중 class가 'abc'인 요소를 선택합니다.
요소[속성!='값'] $("p[class!='abc']") p 요소 중 class가 'abc'가 아닌 요소를 선택합니다.
요소[속성='값'] $("p[class~='abc']") p 요소 중 class가 'abc'인 요소를 선택합니다.
그러나 'abc' 앞 뒤로 연결된 문자가 없어야 합니다. 즉 'main abc'는 선택이 되지만 'main__abc'는 선택되지 않습니다.
요소[속성*='값'] $("p[class*='abc']") p 요소 중 class에 'abc'를 포함하는 요소를 모두 선택합니다. 즉 'main abc', 'main__abc' 모두 선택됩니다.
요소[속성|='값'] $("p[class^='abc']") p 요소 중 class가 'abc' 또는 'abc-'로 시작하는 요소를 선택합니다.
요소[속성$='값'] $("p[class$='abc']") p 요소 중 class가 'abc' 또는 'abc-'로 끝나는 요소를 선택합니다.
