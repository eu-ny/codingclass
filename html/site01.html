SITE 만들기

지금까지 웹페이지에 들어가는 기본적인 유형들을 실습해보았죠? 이번에는 지금까지 배웠던 유형들을 합쳐서 실제 웹페이지 + 반응형을 함께 제작해보겠습니다. 🫠
특히 이번 슬라이더 유형은 직접 자바스크립트를 구현하지 않고 스와이퍼라는 슬라이드 프리소스를 사용하여 제작해보았습니다.

📖 Swiper란?
슬라이드 이펙트를 간단히 작성할 수 있도록 공개된 무료 소스로 HTML, CSS, JAVASCRIPT를 모두 지원해줍니다.

< figma : 디자인 하기 >

제작하기 전 사이트의 구성 요소를 정해줍니다.

#1. HTML

헤더와 푸터 유형을 제외한 유형들은 메인 영역에 섹션별로 나누어 저장하였습니다.
항상 강조했지만 웹사이트를 제작할 땐 웹표준과 접근성, 호환성을 반드시 지키도록 합니다. 특히 사이트에서 skip메뉴 구현은 반드시 구현해주어야 합니다~~!

#3. JAVASCRIPT

헤더와 스크롤 기능, 그리고 슬라이드 구현을 위한 스크립트 입니다.
스와이퍼를 사용하더라도 링크 외에 변수 설정은 따로 해주어야 합니다. 다음은 화살표를 누를 경우 or 터치하여 옆으로 밀 경우 다음 슬라이드로 이동하는 것과 자동으로 2초마다 넘어가도록 변수를 설정해주었습니다.
헤더와 스크롤 기능은 후에 헤더 유형과 함께 추가 될 예정 입니다.

<script src="/경로/swiper-bundle.min.js"></script> : 왼쪽과 같이 HTML에 스와이퍼 프리 소스를 저장하여 따로 넣어주어야 합니다. 

var swiper = new Swiper(".mySwiper", {
    navigation: {
        nextEl: ".swiper-button-next",
        prevEl: ".swiper-button-prev",
    },
    autoplay: {
        delay: 2000,
    },
    pagination: {
        el: ".swiper-pagination",
        type: 'bullets',
    },
});
