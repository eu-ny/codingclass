< CSS 소스 : 스타일 적용하기 >

구조 짜임의 순서대로 css 스타일을 적용시켜 봅시다. 이 때 디자인 시안을 참고하면서 html와 번갈아가며 큰 덩어리 부터 스타일을 적용시켜 나가는 것이 중요합니다.
속성에 대한 내용은 코드와 주석을 보면서 이해해봅시다!

! TIP !
1. display: flex를 줄 때 구조를 잘 파악하여 나누어주어야 합니다.
2. 먼저 헤더부분과 컨텐츠 부분을 나누어 주어 flex를 줍니다.
3. 그 뒤 컨텐츠 부분에서 윗 섹션과 아래 섹션을 나누어서 각 섹션마다 flex를 줍니다.
4. 이미지 스트라이프를 사용할 경우 공통 선택자와 각각의 선택자를 부여합니다.
5. 그 뒤 공통 선택자에게는 url()속성, 각각 선택자에게는 background-position 속성을 사용하여 위치를 조정해줍니다.
