레이아웃04

이번 레이아웃은 전체 영역이 들어간 구조이다. 실제 사이트에서 이런 구조를 많이 사용하며, 컨테이너를 만들어서 가운데 영역을 설정한다.

float을 이용한 레이아웃 + 부모요소 컨테이너 사용방법

아이디 클래스 선택자 선언
id : 한 번 정도 사용하는 것이 좋으며 전체적인 틀, 스크립트에서 많이 사용한다. id로 선언할 경우 후에 고치는 것이 까다롭다.
class : 두개 이상 (중복 가능)사용이 가능하며 후에 고치는 것이 유동적인 편. 주로 세부적인 영역이나 반복적인 것에 많이 사용된다.
