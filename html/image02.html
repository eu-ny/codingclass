< HTML 소스 : 레아아웃 구조 파악하기 >

디자인한 웹 페이지 시안을 보며 페이지 구조를 파악한 후 웹 표준성과 접근성에 맞는시멘틱한 마크업을 해줍니다.

! TIP ! 이번 유형은 백그라운드에 이미지를 삽입하면 따로 애니메이션과 같은 효과를 주기 어렵기 때문에 img태그를 사용하여 이미지를 삽입하여 작업합니다:)
지난 시간에 작업한 이미지 유형01에서 공통 속성을 사용할 수 있도록 섹션에 섹션 선택자를 붙여줍니다.

! html 주의사항 !
• img태그의 속성 중 alt안에 이미지에 대한 설명을 넣어줘야 시각 및 청각 장애인들도 스크린리더기를 통해 이해 가능
• 섹션과 아티클 태그에는 꼭 제목 태그를 자식 요소로 넣어 주어서 웹표준과 웹 접근성, 호환성을 놓치지 않도록 하자!
• a 태그에게 title 속성을 부여해주어야 한다.
• img태그를 사용할 때는 꼭 figure 태그를 선언해주어야 한다.

< CSS 소스 : 스타일 적용하기 >

속성을 지정할 때는 디자인 시안을 참고하면서 html와 번갈아가며 큰 덩어리 부터 스타일을 적용시켜 나가는 것이 중요합니다.
transition, transform, backdrop-filter를 통해 동적인 스타일 속성을 주도록 합시다!
속성에 대한 내용은 코드와 주석을 참고하시면 됩니다:)

TIP !
1. 먼저 전체 초기화(Reset)와 폰트 디폴트 설정을 해주는 것이 좋다.
2. 컨테이너 설정을 통해 사이트의 크기와 여백을 설정해준 뒤 메인 콘텐츠인 섹션에게 스타일을 적용시킨다. : 이렇게 컨테이너로 작업하게 되면 더 효율적으로 속성을 적용시킬 수 있게 된다.
3. :hover 속성을 통해 커서가 해당 요소에 닿았을 때의 동적 효과를 주도록 한다.

    justify-content: space-between;

    vertical-align: top; /* 수직 정렬 */

}
.image__desc {
    box-sizing: border-box;
    width: 100%;
    height: 100px;
    text-align: center;
    position: absolute;
    left: 0;
    bottom: 0;
    backdrop-filter: blur(10px);    /* 블러 효과 */
    padding: 23px 20px;
    bottom: -100px;
    transition: all 0.3s ease-in-out;    /* 움직임 속도 효과 : 전체를 0.3초 동안 천천-보통-천천 순으로 움직이게 하기 */
}
.image:hover .image__desc {      /* 커서를 갖다 대었을 때 desc가 위로 올라오도록 설정 : 요소 안에 들어오기 때문에 오버플로우 히든 속성을 받지 않음*/
    bottom: 0;     
}
.image:hover .image__box img {
    transition: all 0.6s ease-in-out;    /* 움직임 속도 설정 */
    transform: scale(1.05);      /* 이미지를 확대 시키기 */

! Css 속성 알아가기 !
transition : 움직임 속도 설정
transform : 요소의 모양, 크기, 위치 등을 자유롭게 변경
backdrop-filter : 배경에 그래픽 효과를 설정 (배경에 투명도가 있어야 효과를 확인할 수 있다.)
