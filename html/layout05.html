레이아웃05

전체 영역이 들어간 구조로 세부적인 요소들은 부모컨테이너를 만들어서 가운데 영역으로 레이아웃을 설정한다.
이번 레이아웃은 좀 더 심화과정으로 들어간다.

float을 이용한 반응형 레이아웃

CSS 스타일을 효율적으로 주기 위해서는 html 마크업을 깔끔하게 선언할 줄 알아야 한다. 때문에 먼저 레이아웃 분석 및 요소의 이름을 잘 선언하는 것이 중요하다.

clearfix를 이용한 float의 영역깨짐 방지
1. 깨지는 영역에 clear:both를 설정한다.
2. 부모 박스 영역에 overflowf: hidden을 설정한다. - 애니메이션 효과가 적용되지 않는다.
3. clearfix를 설정한다. - 깨진 레이아웃 요소의 부모요소에게 클래스로 이름을 선언해준다.

gird를 이용한 반응형 레이아웃

세부적인 요소들이 복잡하게 들어가 있는 경우 효율적인 마크업과 시멘틱 마크업에 대한 중요도가 높아진다.
각 요소에 대한 부모요소와 자식요소들을 효율적으로 분석하고 나눌 줄 알아야 한다. 

 
* {
    margin: 0;
}

#wrap {}

#header {
    height: 100px;
    background-color: #EEEBE9;
}

#nav {
    height: 100px;
    background-color: #BCAAA4;
}

#main {
    height: 780px;
    background-color: #886F65;
}

#footer {
    height: 100px;
    background-color: #4E342E;
}

/* 컨테리너 설정 */

.container {
    width: 1200px;
    height: inherit;
    margin: 0 auto;
    background-color: rgba(0, 0, 0, 0.3);
}

/* contents 설정 */

.contents {
    display: grid;
    grid-template-areas:
        "cont1 cont1"
        "cont2 cont2"
        "cont3 cont4"
    ;
    grid-template-columns: 50% 50%;
    grid-template-rows: 100px 200px 480px;
}

.contents .cont1 {
    background-color: #74574A;
    grid-area: cont1;
}

.contents .cont2 {
    background-color: #684D43;
    grid-area: cont2;
}

.contents .cont3 {
    background-color: #594139;
    grid-area: cont3;
}

.contents .cont4 {
    background-color: #4A352F;
    grid-area: cont4;
}

/* 미디어쿼리 */

@media (max-width:1220px) {
    .container {
        width: 96%;
    }

    .contents {
    display: grid;
    grid-template-areas:
        "cont1 cont2 cont2"
        "cont1 cont3 cont4"
    ;
    /* grid-template-columns: 1fr 1fr 1fr;  3등분 */
    grid-template-columns: repeat(3, 1fr);  /* 3등분 */
    /* grid-template-columns: repeat(3, 33.3333%);  /* 3등분 */
    /* grid-template-columns: 33.3333% 33.3333% 33.3333%;  3등분 */
    /* grid-template-rows: 780px 200px 480px; */
    grid-template-rows: repeat(2, 1fr);
    }
}

@media (max-width:768px) {
    .contents {
    display: grid;
    grid-template-areas:
        "cont1 cont2"
        "cont1 cont3"
        "cont1 cont4"
    ;
    grid-template-columns: 30% 70%;
    grid-template-rows: repeat(3, 1fr);
    }
}   

@media (max-width:480px) {
    .contents {
    display: grid;
    grid-template-areas:
        "cont1"
        "cont2"
        "cont3"
        "cont4"
    ;

    }
