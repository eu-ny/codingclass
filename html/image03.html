! html 주의사항 !
• img태그의 속성 중 alt안에 이미지에 대한 설명을 넣어줘야 시각 및 청각 장애인들도 스크린리더기를 통해 이해 가능

• img태그를 사용할 때는 꼭 figure 태그를 선언해주어야 한다.

< CSS 소스 : 스타일 적용하기 >

이번 레이아웃은 flex가 아닌 그리드 기법으로 작업해보겠습니다.
transition, transform, backdrop-filter를 통해 동적인 스타일 속성을 주도록 하고 이미지 스프라이트 기법을 통해 아이콘을 효율적으로 삽입해보도록 합시다!
속성에 대한 내용은 코드와 주석을 참고하시면 됩니다:)

TIP !
1. 먼저 전체 초기화(Reset)와 폰트 기본 설정을 해주는 것이 좋다.
2. 큰 섹션부터 속성을 적용시킨다. 세부적인 요소는 마지막에 적용한다.
3. :hover 속성을 통해 커서가 해당 요소에 닿았을 때의 동적 효과를 주도록 한다.
4. :nth-child( ) 속성을 사용하여 이미지 스프라이트와 그리드 레이아웃을 효율적으로 작업할 수 있다.

! Css 속성 알아가기 !
transition : 움직임 속도 설정
transform : 요소의 모양, 크기, 위치 등을 자유롭게 변경
backdrop-filter : 배경에 그래픽 효과를 설정 (배경에 투명도가 있어야 효과를 확인할 수 있다.
:nth-child( ) : n번째 자식 요소에게만 스타일 적용
background: url( ) : 백그라운드에 이미지 삽입
