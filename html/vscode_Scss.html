🔶 02. Live Sass Compiler

아래의 Live Sass Compiler는 실시간으로 SCSS를 CSS로 컴파일 시켜주는 확장 프로그램입니다.

다운 모두 받으셨나요?
그럼 다음의 방법과 함께 차근차근 적용시켜보겠습니다~!

1. 먼저 cmd + ⇧ + p 또는 F1을 눌러서 'json'을 검색한 후 'settings.json'에 들어갑니다.
(들어가는 방법은 지난 에러 솔루션을 참고하시면 됩니다~! )

2. 그 다음 다음의 코드를 중괄호 안에 넣어주면 됩니다. 

 "liveSassCompile.settings.generateMap": false,
"liveSassCompile.settings.formats": [
    {
    "format": "expanded",
    "extensionName": ".css",
    "savePath": "~/../css/"
    },
    {
    "format": "compressed",
    "extensionName": ".min.css",
    "savePath": "~/../css/"
    }
]

2-1. 우선 위의 코드가 어떤 의미인지 간단히 이해해보겠습니다. 

"liveSassCompile.settings.generateMap": false,

컴파일 할 경우 자동으로 MAP파일과 정보들이 함께 표기되는데 이를 해제하려면 false로 바꿔주시면 됩니다. 

"liveSassCompile.settings.formats": [
    {
    "format": "expanded",
    "extensionName": ".css",
    "savePath": "~/../css/"
    },
    {
    "format": "compressed",
    "extensionName": ".min.css",
    "savePath": "~/../css/"
    }
]

포멧설정, 컴파일 했을 때 css 파일의 성격 ( .css: 일반 css 파일 / .min.css : 축약 형식의 css )
"savePath": "~/../css/"는 새로 css폴더를 생성하여 그곳에 저장하겠다는 의미입니다.

3. 아래의 빨간 박스에 위치한 버튼을 클릭하여 작동이 잘 되는 지 테스트하면 됩니다!
